# AXM Readiness acceptance assets

Readable source for the production Pages surface at `/acceptance.html`.

- `app.css` contains the interface styles.
- `app-0.js` through `app-5.js` are ordered classic-script modules.
- `build_offline.py` emits a self-contained `AXM_Readiness_Offline.html` during the Pages workflow.

The browser application performs local-only intake analysis. It does not mint AXM Genesis shards or make outbound network requests.
