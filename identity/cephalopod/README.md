# Cephalopod icon research v2

**Status:** exploratory. No candidate is canonical or in custody.

Version 2 corrects two failures in the first package. The source residue records the visible construction habits of the referenced sprite sheets and SimCity screen assets rather than merely summarizing the design conversation. The proof matrix is also designed against compositional collapse. Its sixteen candidates vary body-to-arm ratio, orientation, negative-space strategy, substrate topology, corruption mechanism, and palette. The right side is never treated as a brick wall.

## Durable source material

- `index.html` is the dependency-free review surface. It renders the adjacent atlas and exact map ledger without a build step.
- `SOURCE_RESIDUE_ATLAS.svg` is an original visual trace of source-specific construction habits. No third-party sprite pixels are reproduced.
- `variants-v2.rle.json` contains exact run-length-encoded maps for all 16 candidates at 32×32, 24×24, 16×16, and 12×12.
- `RESEARCH.md` records source-specific observations, transferable construction residue, and rejected character residue.
- `SOURCES.json` records page metadata and the capture boundary.
- `PROOF_MATRIX.md` describes the four controlled searches and all sixteen hypotheses.
- `NOTICE.md` states the third-party reference boundary.

## Control test

A candidate passes only when the organism remains coherent and gains a visible route, contact, brace, or opening because the substrate has entered a structured console-style failure state. A candidate fails when it resembles an octopus attacking a vertical obstacle, when the body itself is smeared into glitch debris, or when palette changes are the only meaningful variation.
